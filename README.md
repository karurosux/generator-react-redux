# reatux [![NPM version][npm-image]][npm-url] [![Build Status][travis-image]][travis-url] [![Dependency Status][daviddm-image]][daviddm-url]

> Generate react project with redux.

## Installation

```bash
npm -g install reatux
```

for now run

```
reatux --help
```

or

```
rtx --help
```
